Studied the Cantor Set using concepts from Fractal Geometry and Topology.

Analyzed the geometric construction of the Cantor set through the iterative removal of segments from the unit interval [0,1].

Explored the concept of Box-Counting Dimension to determine the fractional (non-integer) dimension of the Cantor set.

Investigated key properties such as uncountability, zero measure, and fractal structure.

Developed three generalized versions of the Cantor set based on different removal processes.

Examined the fractal dimensions of generalized Cantor sets and compared their structural properties.

Applied the Cantor set framework to analyze chaotic dynamics generated by the iteration of a piecewise linear map.

Demonstrated connections between fractal structures and chaos in dynamical systems.
